PMSP Torch
==========

Installation
------------

::

    git clone https://projects.sisrlab.com/cap-lab/pmsp-torch
    make requirements
    make test

Or install straight from git
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

::

    pip install git+https://projects.sisrlab.com/cap-lab/pmsp-torch@master
    pmsp.py test
    pmsp.py simulate

Running simulation
------------------

::

    make run

Viewing stimuli
---------------

::

    make view

Writing stimuli to a file
-------------------------

::

    make write
