PMSP Torch
==========

Installation
------------

::

    git clone https://projects.sisrlab.com/cap-lab/pmsp-torch
    make requirements
    make test

::

    pip install git+https://projects.sisrlab.com/cap-lab/pmsp-torch@master

Running simulation
------------------

::

    make run

Viewing stimuli
---------------

::

    make view

Writing stimuli to a file
-------------------------

::

    make write
